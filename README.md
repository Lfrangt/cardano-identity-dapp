# CardanoID - Decentralized Identity DApp

A beautiful, Apple-inspired decentralized identity platform built on Cardano blockchain.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open your browser and visit
http://localhost:3000
```

## ✨ Features

- **Apple-inspired Design**: Beautiful, modern interface with glassmorphism effects
- **Decentralized Identity**: Built on Cardano blockchain for true data ownership
- **Social Connections**: Connect with like-minded people based on interests and skills
- **Reputation System**: Build your reputation through verified endorsements
- **Privacy Controls**: Granular privacy settings for different types of information

## 🛠️ Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS with custom Apple-inspired design system
- **Animations**: Framer Motion for smooth interactions
- **Icons**: Lucide React
- **Blockchain**: Cardano (Plutus smart contracts)
- **Storage**: IPFS for decentralized data storage

## 🎨 Design System

The project uses an Apple-inspired design system with:

- **Typography**: SF Pro Display inspired font stack
- **Colors**: Blue to purple gradients with careful gray scales
- **Spacing**: Consistent spacing system based on Apple's design principles
- **Shadows**: Subtle shadows and glassmorphism effects
- **Animations**: Smooth, natural animations using cubic-bezier curves

## 🔗 Smart Contracts

The DApp integrates with Cardano smart contracts for:

- **Identity Registry**: Store and manage user profiles
- **Social Graph**: Handle friend connections and relationships
- **Reputation System**: Track and verify user endorsements
- **Token Economics**: Reward users for platform participation
